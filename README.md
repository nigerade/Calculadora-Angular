# Calculadora-Angular

Projeto criado em Angular pelo curso Técnico Desenvolvimento de Sistemas pela Etec Albert Einstein.
O usuário consegue entrar com 2 Valores e usar as Funções: Somar, Subtrair, Multiplicar e Dividir.
